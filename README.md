# Confessions


### Requirements

- Git

- Nodejs v14+

- Visual Studio Code

- Chrome


### Prerequisite

Everyone should have `yarn` installed globally.
To install yarn, run -
```
npm i -g yarn
```


### To Run Locally

- Clone this repo

- `cd` into the project folder

- Install dependencies
  ```
  yarn install
  ```

- To start bundler, run
  ```
  yarn start
  ```
